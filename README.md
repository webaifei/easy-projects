# easy project
> 轻松管理你的项目

```
project -h
project -V --version
project set config=config.js
project editor list 
project editor default
project open project_name -e=webstorm

project list

```

## depedencies
1. chalk
2. Inquirer.js
3. commanderjs or yargs
4. figlet
5. shelljs



### 功能
1. 项目展示 ✔️
2. 项目选择
3. 项目打开
4. 配置 
5. 打开前端工具箱
  - 测试环境：根据手机号查询 token 51token userid
  - 拼链接
  
6. 网址导航
  - 项目wiki
  - 前端常用地址
